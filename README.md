# Funny Sandboxes

Small interactive sandboxes built to make tail risk and leverage intuition tangible. Each folder is a self-contained toy lab with its own README.

**[Browse online →](https://paul-engerran.github.io/funny-Sandboxes/)**

## Labs

| Lab | What it does |
|-----|-------------|
| [leverage-fat-tails/](./leverage-fat-tails/) | Liquidation cascades produce fat tails from Gaussian shocks. Monte Carlo + single path explorer. |

## Scope

Toy models for explanation and teaching. Not identification strategies. Not for trading.
